3. TEXT EDITOR

A text editor is a program that lets you edit and modify text files. It is often used for writing and editing code, documents, and other types of plain text files. A good text editor should have features like syntax highlighting, code completion, and a comfortable interface.

A screen reader is simply a program that does some analysis of the text that appears on a screen, or in a window, and passes it to a text-to-speech synthesizer. Most screen readers are phenomenally unintelligent about their analysis of what is on the screen.
